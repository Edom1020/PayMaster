💼 PayMaster - Sistema de Nómina

📌 Descripción
PayMaster es una aplicación desarrollada en Java que permite gestionar y calcular el salario de diferentes tipos de empleados. El sistema aplica principios de Programación Orientada a Objetos y buenas prácticas de desarrollo como SOLID y pruebas unitarias.

---

⚙️ Funcionalidades

- Registro de empleados:
  - Asalariados
  - Por horas
  - Por comisión
  - Temporales
- Cálculo de salario bruto
- Cálculo de salario neto (con deducciones)
- Interfaz gráfica para interacción con el usuario
- Pruebas unitarias con JUnit

---

🛠️ Tecnologías utilizadas

- Java
- Swing (Interfaz gráfica)
- Maven
- JUnit
- Git & GitHub
